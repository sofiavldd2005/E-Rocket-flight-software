# E-Rocket Flight Software

Welcome to the E-Rocket flight software repository. This package implements a ROS 2 based flight stack designed to interface with PX4 via the MicroXRCEAgent.

## Documentation
- **[Architecture Overview](../../Documentation/architecture.md)**: Explains the ROS 2 node graph, the fast/slow loop logic, and how the PX4 communication is handled.
- **[Launch Guide](../../Documentation/launch_guide.md)**: Explains the different launch scripts, how to start the software for hardware flights, and how to run Software-in-the-Loop (SITL) simulations.
